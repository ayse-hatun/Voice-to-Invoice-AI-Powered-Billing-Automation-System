# 🎙️ Voice to Invoice

### AI-Powered Billing Automation System

</div>

<p align="center">
A voice-driven intelligent billing system that transforms speech into structured invoices using AI, enabling hands-free financial management for modern businesses.
</p>

---

## 🚀 Live System Overview

- 🎤 Speak invoice details naturally  
- 🧠 AI extracts structured billing data  
- 📊 Auto-syncs to database in real time  
- 📑 Generates clean invoice records instantly  

---

## ⚡ Core Features

✔️ Voice-to-text transcription (Whisper / speech models)  
✔️ AI-powered data structuring (clients, items, totals)  
✔️ Automatic invoice generation  
✔️ Real-time database sync (Supabase)  
✔️ CashFlow Dashboard (live financial overview)  
✔️ Authentication + signup flow  
✔️ Responsive UI with animated experience
